This is a Group Assignment Project for the Advanced Programming Module done by Chew-Yen Mei, Clement Liew, Frank Penner and Yuen Tze Svenn. It is about a chatbot system named S.T.E.V.E that can do the following features:

1. Course Q&A  
-Student asks questions  
-System retrieves course materials by RAG 

2. Chapter and Topic Summary  
-Automatically summarises the course materials  
-Highlight key points 
-Generate revision notes 

3. Code Explanation 
-Generate MCQs, True/False, and short answer questions based on RAG-retrieved course material.  
-Provide correct answers along with explanations. 

4. Quiz Generation  
-Explain Java code. 
-Fix minor errors. 
-Provide step-by-step reasoning.

Lecturers can also log into the system to upload files for the chatbot to use to answer questions asked by students.
